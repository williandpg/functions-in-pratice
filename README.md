<h1><strong>Functions in Practice</strong></h1>

<h2><strong>Descrição</strong></h2>
<p align="justify">
  Este repositório contém um exercício prático de lógica de programação com foco na criação e estruturação de funções JavaScript. O objetivo é demonstrar como responsabilidades bem definidas tornam o código mais claro, reutilizável e fácil de manter.
</p>

<h2><strong>Tecnologias Utilizadas</strong></h2>
<ul>
  <li>
    <a href="https://developer.mozilla.org/pt-BR/docs/Web/JavaScript" target="_blank">
      <strong>JavaScript</strong></a>: Linguagem de programação utilizada para criar lógica, manipular dados e estruturar funções com responsabilidades bem definidas.
  </li>

  <li>
    <a href="https://nodejs.org/" target="_blank">
      <strong>Node.js</strong></a>: Ambiente de execução que permite rodar código JavaScript no back-end e executar os testes localmente no terminal.
  </li>
</ul>


<h2><strong>Estrutura do Projeto</strong></h2>
<p align="justify">
  A estrutura do projeto é organizada para facilitar a navegação e o entendimento do código. Abaixo está a estrutura de diretórios e arquivos:
</p>

<pre><code>
/
├── src/
│   └── <código fonte>

├── package.json

</code></pre>
